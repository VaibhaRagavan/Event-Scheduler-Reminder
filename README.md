

# 📅 Event Scheduler & Reminder (Serverless)

A serverless event scheduler that allows users to create, delete, and view events with reminders.  
Built with **AWS Lambda, API Gateway, and a simple frontend**.

---

## 🚀 Features

- 🗓️ Create, update, and delete events
- 🔔 Automatic reminders (via email)
- 💻 Web interface (HTML/JS)
- ⚡ Fully serverless backend with AWS Lambda + API Gateway

---

## 🛠️ Tech Stack

- **Backend**: Python AWS Lambda functions  
- **Frontend**: HTML, JavaScript  
- **Serverless Infrastructure**: AWS SAM / CloudFormation  
- **Data Storage**: Optional JSON file or DynamoDB
